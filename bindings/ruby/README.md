# OpenDAL Ruby Binding

This crate intends to build a native ruby binding.

![](https://github.com/apache/incubator-opendal/assets/5351546/87bbf6e5-f19e-449a-b368-3e283016c887)

## Development

Install gems:

```shell
bundle
```

Build bindings:

```shell
rake compile
```

Run some tests:

```shell
rake test
```
