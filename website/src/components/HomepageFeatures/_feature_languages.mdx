Apache OpenDAL provides [Rust Core](pathname:///docs/rust/opendal/) and builds different language bindings like [Node.js Binding](pathname:///docs/nodejs/) and [Python Binding](pathname:///docs/python/).

> *More bindings like [C](https://github.com/apache/incubator-opendal/blob/main/bindings/c/README.md), [Java](https://github.com/apache/incubator-opendal/blob/main/bindings/java/README.md), [Ruby](https://github.com/apache/incubator-opendal/blob/main/bindings/ruby/README.md) are still being worked on.*
